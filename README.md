
# 🗂️ Desktop Organizer от Людмил Тончев

**Desktop Organizer** е полезен инструмент, създаден с Python и Tkinter, който помага да поддържате вашия десктоп чист и организиран с един клик. Край на хаоса от икони!

## ✨ Функции
- **Организиране по категории:** Автоматично разпределя файловете в папки (Изображения, Документи, Видео и др.).
- **Организиране по дата:** Групира файловете според годината и месеца на създаване.
- **Организиране по разширение:** Създава папка за всеки конкретен тип файл (напр. .pdf, .jpg, .docx).
- **Безопасност:** Опция за създаване на резервно копие (Backup) преди започване.
- **Графичен интерфейс:** Лесен за използване интерфейс, подходящ за всеки.

## 🚀 Как да използвате програмата
Имате два варианта:

### 1. За обикновени потребители (EXE)
- Отидете в секцията [Releases](линк_към_твоя_release) вдясно.
- Изтеглете файла `desktop_organizer.exe`.
- Стартирайте го и изберете метод на организиране.

### 2. За разработчици (Python)
Ако искате да стартирате кода директно:
